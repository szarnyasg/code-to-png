# code-to-png

## Prerequisites

Rather a lot:

* TeX Live
* Pygments
* Imagemagick

```bash
sudo apt-get install -y texlive-full
sudo apt-get install -y python-pygments
sudo apt-get install -y imagemagick
```

## Guide

To run, issue:

```
make clean build
```

## Limitations

Your students can still use [OCR](https://en.wikipedia.org/wiki/Optical_character_recognition).
